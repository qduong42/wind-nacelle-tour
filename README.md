# Inside the Nacelle — Interactive Wind Turbine Tour

An educational 3D cutaway of a geared wind-turbine nacelle. Orbit the machine,
click any of the nine numbered components, watch the drivetrain spin at its real
~1:100 gear ratio, and x-ray the shell or individual housings.

Each component card explains what the part is, what it does, why it matters in
operations, and which SCADA signals it produces — including where each sensor
physically sits in the machine.

Built with Three.js (fully procedural geometry, no external assets) and shipped
as a single self-contained `index.html`. Deployed as a static site via
`render.yaml`.
